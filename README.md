# Portfolio (template)

A minimal Django portfolio template: one-page layout with about, experience, projects, and articles. Edit `core/site_data.py` to add your content.

**Run locally:** see [RUNNING.md](RUNNING.md) for steps on Windows and macOS.

---

Design inspired by [Brittany Chiang](https://brittanychiang.com).

The original portfolio was designed using Figma and built with Next.js and Tailwind CSS.
This project adapts a similar layout and structure, implemented with Django.
